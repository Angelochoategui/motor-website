#### A Navigation item 

## Usage

A Nav item to navigate to different pages. Recommended for use in the Sidebar component of a multi-page app.

```javascript

import { NavItem } from @motor-js/core
//...
    <NavItem
      path="/sales"
      name="Executive Sales"
      css="fa fa-fw fa-home"
      key="2"
    />

```
