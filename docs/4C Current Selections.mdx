#### Current Selections Box. Returns the current app selections.


## Usage

```javascript

import { CurrentSelections } from @motor-js/core

//...

<CurrentSelections />

```


## Props

:::info
You do not need to set the Config prop if the component is a child of the Motor component
:::

 
<table>
  <tr>
    <th>Prop</th>
    <th>Description</th>
    <th>Options / Example</th>
  </tr>
  <tr>
    <td><code>config</code></td>
    <td>Configuration object to connect to the Qlik Engine, only set this if used outside of the Motor component</td>
    <td></td>
  </tr>
 
</table>

## Examples

##### Simple Filter example

```jsx live

function SelectionsTest() {
    return (
    <Motor config={config}>  
     <CurrentSelections />
    </Motor>
    )
}

```
