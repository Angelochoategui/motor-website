## Overview

The Motor JS provides a base color system which can all be edited to match your brand colors and style. 
All of the colors used in the framework are defined in our theme object, within global.color.

These colors can be both edited, by editing the theme object and referenced from all of our components. 
See examples of this below

## Open Color

Motor JS come bundled with Open Color (https://yeun.github.io/open-color/) an open source color scheme.

These are all stored in variables which can be accessed throughout the framework. See below for an example of how to reference
the range of blues from open color.

### KPI using a color variable from Open Color

```jsx live
function ColumnDemo() {
  return (
    <Motor config={config}>
      <KPI
        color='var(--oc-blue-5)'
        label="My Blue KPI"
        cols={["=100"]}
      />
    </Motor>
  );
}
```
