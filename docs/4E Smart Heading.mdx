#### Smart Heading, use as free test heading, or assign as the app name or last reload time.

## Usage

```javascript

import { SmartHeading } from @motor-js/core

//...
<SmartHeading />

```

## Props

:::info
You do not need to set the Config prop if the component is a child of the Motor component

## Examples

##### Simple Smart Heading example

```jsx
function SmartHeadingTest() {
  return (
    <Motor config={config}>
      <SmartHeading />
    </Motor>
  );
}
```
