#### A Filter. Make selections and filter your data.


## Usage

```javascript

import { Filter } from @motor-js/core

//...

<Filter cols=['Dimension Name'] />

```


## Props

:::info
You do not need to set the Config prop if the component is a child of the Motor component
:::

  <tr>
    <td><code>size</code></td>
    <td>Size of the filter</td>
    <td>'tiny'< br/>'small'< br/><b>'medium'</b>< br/>'large'< br/>'xlarge'</td>
  </tr>
  <tr>
    <td><code>width</code></td>
    <td>width of the filter</td>
    <td>'string'<br/><b>'100%'</b></td>
  </tr>

    <td><code>dropHeight</code></td>

    <td><code>margin</code></td>

    <td>'string'<br/><b>'5px'</b></td>
  </tr>

    <td><code>onSelectionChange</code></td>

    <td><code>onSearch</code></td>
    <td>function fired on search</td>
