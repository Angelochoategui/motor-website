#### A Filter. Make selections and filter your data.


## Usage

```javascript

import { Filter } from @motor-js/core

//...

<Filter cols=['Dimension Name'] />

```


## Props

:::info
You do not need to set the Config prop if the component is a child of the Motor component
:::
