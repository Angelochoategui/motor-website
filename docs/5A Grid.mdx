#### Grid Layout. A CSS Grid Layout Component

## Usage

This is a Grid layout component based off CSS Grid. It is a 2-dimensional layout system,
aimed at making it easy to structure your Dashboard

```javascript

import { Grid } from @motor-js/core
//...
<Grid
  rows={["60px", "auto", "30px"]}
  cols={["auto"]}
  areas={[["header"], ["main"], ["footer"]]}
  backgroundColor="altGray1"
>
  //...
</Grid>

```

## Sample Syntax

### Grid configuration

The below syntax creates a grid of three rows and one column.
The rows have been defined in the area property as header, main and footer.

```jsx {11}
//...
<Grid
  rows={["60px", "auto", "30px"]}
  cols={["auto"]}
  areas={[["header"], ["main"], ["footer"]]}
  backgroundColor="altGray1"
>
  ...
</Grid>
```
