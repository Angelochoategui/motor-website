#### A Button. Trigger various Qlik actions such as clearing selections, or attach a custom event.


## Usage

```jsx live

function ButtonTest() {
    return (
    <Juno config={config}>  
      <Button backgroundColor='brand'>I am a Button.</Button>
    </Juno>
    )
}

```
